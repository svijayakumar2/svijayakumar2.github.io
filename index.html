Research

Grounding Neural Inference with Satisfiability Modulo Theories

NeurIPS 2023 (Spotlight)
Zifan Wang*, Saranya Vijayakumar*, Kaiji Lu, Vijay Ganesh, Somesh Jha, Matt Fredrikson

Paper Code Talk Video + Slides

We propose a novel framework for enhancing neural networks with SMT solvers to improve their reasoning capabilities while maintaining efficiency. Key contributions:

  • A neural-symbolic architecture that combines transformers with SMT solving
  • Theoretical analysis showing completeness guarantees for certain reasoning tasks
  • Empirical evaluation showing 15% improvement in accuracy on complex reasoning tasks

Technical Details:
Our method introduces a differentiable interface between neural networks and SMT solvers, allowing end-to-end training while leveraging logical reasoning capabilities. The architecture consists of:

  • A transformer encoder that processes input problems into logical constraints
  • A novel differentiable SMT layer that performs logical reasoning
  • A neural decoder that converts logical solutions back to natural language

Results:
Evaluated on mathematical reasoning benchmarks, our approach achieves:

  • 92.5% accuracy on ProofWriter dataset (15% improvement over SOTA)
  • 85.3% accuracy on LogicInference benchmark
  • Maintains inference time within 2x of pure neural approaches

CallMine: Fraud Detection and Visualization of Million-Scale Call Graphs

CIKM 2023
Mirela Cazzolato, Saranya Vijayakumar, Meng-Chieh Lee, Namyong Park, Catalina Vajiac, Christos Faloutsos

Paper Code

We present CallMine, a system for detecting and visualizing fraud patterns in large-scale telecommunication networks. The system combines advanced graph mining techniques with interactive visualizations to help investigators analyze millions of calls efficiently. Key contributions:

  • Novel algorithms for detecting suspicious patterns in call graphs
  • Interactive visualization system for exploring million-node graphs
  • Real-world deployment and case studies with telecom providers

Technical Details:
CallMine introduces several technical innovations:

  • FastRP-based graph embeddings for anomaly detection
  • Temporal pattern mining using sliding windows
  • Multi-level visualization system with semantic zooming

Results:
Evaluation on real-world telecom data shows:

  • Processing of 1M+ node graphs in under 5 minutes
  • 90% precision in detecting known fraud patterns
  • Successfully deployed with multiple telecom providers
